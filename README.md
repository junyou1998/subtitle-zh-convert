# ZhConvert 字幕轉換器 (ZhConvert Subtitle Converter)

[![Deploy](https://img.shields.io/badge/Deploy-zhconvert.junyou.tw-blue?style=flat-square&logo=vercel)](https://zhconvert.junyou.tw)

這是一個基於 [繁化姬 (ZhConvert)](https://zhconvert.org) API 開發的純前端字幕轉換工具。旨在提供一個現代化、直覺且便捷的介面，協助使用者批次將字幕檔案進行繁簡轉換與本地化處理。

🔗 **線上使用**：[https://zhconvert.junyou.tw](https://zhconvert.junyou.tw)

## ✨ 主要功能

*   **批次轉換**：支援同時上傳多個 `.ass` 或 `.srt` 字幕檔案進行處理。
*   **多種轉換模式**：支援台灣化、香港化、簡體化、中國化等多種轉換模式。
*   **日文保護機制**：可設定保護日文不被轉換，或僅保護原文與日文相同的字。
*   **進階模組支援**：整合繁化姬強大的詞語模組（如：修正錯別字、動漫專有名詞轉換等）。
*   **差異比較**：提供轉換前後的左右對照視窗，並以顏色標示新增與刪除的內容，方便校對。
*   **檔名轉換**：可選擇是否連同檔名一起進行轉換。
*   **深色/淺色模式**：支援系統跟隨、深色與淺色主題切換，提供舒適的閱讀體驗。
*   **純前端處理**：所有檔案操作皆在瀏覽器端完成，不會將您的檔案儲存至我們的伺服器。

## �️ 技術棧

本專案採用以下現代化前端技術構建：

*   **核心框架**：[Vue 3](https://vuejs.org/) (Composition API + Script Setup)
*   **開發工具**：[Vite](https://vitejs.dev/) + [TypeScript](https://www.typescriptlang.org/)
*   **樣式設計**：[Tailwind CSS 4.0](https://tailwindcss.com/)
*   **API 串接**：[Axios](https://axios-http.com/)
*   **檔案處理**：[JSZip](https://stuk.github.io/jszip/) (用於批次打包下載)
*   **圖示庫**：[Lucide Vue](https://lucide.dev/)

## �🚀 操作說明

1.  **上傳檔案**：
    *   將您的 `.ass` 或 `.srt` 字幕檔拖曳至主畫面的上傳區，或點擊上傳區選擇檔案。
2.  **基礎設定**：
    *   選擇您要轉換的目標模式（預設為「台灣化」）。
    *   設定日文部分的處理策略（預設為「保護」）。
    *   若需要轉換檔名，請勾選「轉換檔名」選項。
3.  **進階設定 (選用)**：
    *   點擊「進階設定」按鈕，勾選您需要的優化模組（例如：精靈寶可夢、標點符號修正等）。
4.  **開始轉換**：
    *   確認設定無誤後，點擊「開始轉換」按鈕。系統將透過 API 進行批次處理。
5.  **檢視與下載**：
    *   轉換完成後，可點擊列表中的「📄」圖示查看差異比較。
    *   點擊「📥」圖示下載單個檔案，或點擊上方的「下載選取」按鈕批次下載所有檔案（ZIP 格式）。
